### 关键参数

* 150MHz Cortex®-M33
* 64KB SRAM + 512KB Flash + SDRAM(2MB/8MB/16MB)
* 1个RGB565接口(64Pin)，1个RGB888接口(100Pin)
* JPEG硬件解码器，DMA2D
* USB OTG*1, I2C*2, SPI*2, UART*4, I2S*2, CAN*2, SDIO*1
* 5个32位高级定时器
* 3个比较器，4个运放，1个DAC
* 2个12位19通道的SAR ADC
* 工作电压2.0V - 3.6V,多种省电工作模式
* 提供 QFP64/QFP100 封装
* 工作温度范围（环境温度）-40℃ - 85℃
