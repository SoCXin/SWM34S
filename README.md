#### 关键特性

* CORDIC三角函数计算模块，可快速计算sin、cos等三角函数；
* 16位高精度SIGMA-DELTA ADC，前置7级可编程增益，最大32倍；
* CAN总线接口，支持DMA传输，最大速率1Mbit/s；
* 4COM*32SEG SLCD模块，最大支持128段LCD显示屏。
